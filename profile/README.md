# Kiloloop

Open-source tools and protocols for multi-agent engineering workflows.

## Projects

- **[OACP](https://github.com/kiloloop/oacp)** — Open Agent Coordination Protocol. A file-based coordination protocol that enables AI agents on different runtimes to collaborate asynchronously through a shared filesystem.
- **[oacp-skills](https://github.com/kiloloop/oacp-skills)** — Reusable skills for OACP-compatible agents. Check-inbox, review-loop-reviewer, and review-loop-author form a complete agent-to-agent code review loop.
- **[cortex](https://github.com/kiloloop/cortex)** — Reference app built on OACP. Multi-agent SSOT (single source of truth) that consolidates session debriefs into a daily snapshot.
